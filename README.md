# visit-frontend

Visit frontend scaffold initialized with `npm create vite@latest`.

Current state: intentionally empty application shell.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Backend integration points

- `POST /api/v1/visit/lead`
- `GET /api/v1/visit/content`

Client helpers are prepared in `src/api.js`.
